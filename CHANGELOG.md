# Change Log

All notable changes to the "the-mango-theme" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Unreleased]

## [0.1.0] - 2026-05-31

### Added

- Full Atom One Dark workbench colors: sidebar, activity bar, tabs, status bar, terminal, panels, lists, and more
- Semantic highlighting support for constants, enum members, and default library symbols
- Mango accent color on active tabs, focus borders, badges, and progress bar

### Changed

- Editor background updated from `#263238` to Atom's `#282c34`
- Editor foreground and syntax colors aligned to Atom One Dark palette (`#abb2bf`, `#c678dd`, `#98c379`, `#61afef`, etc.)
- Punctuation and operators now use Atom's neutral gray instead of cyan
- Variables use Atom's red (`#e06c75`); `this`/`self` use yellow (`#e5c07b`)

## [0.0.3]

- Initial release with basic syntax highlighting
